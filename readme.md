# Hercules: Gesture-Controlled Virtual OS

A high-performance Human-Computer Interaction (HCI) system using **MediaPipe** for hand tracking, **OpenCV** for image processing, and **MySQL** for event logging.

## 🚀 Features
* **Virtual Mouse:** Index finger navigation with Recursive Low-Pass Filtering (Smoothing).
* **Dynamic Volume Control:** Vertical displacement tracking for volume adjustment.
* **SQL Logging:** Every action is logged to a relational database for audit trails.
* **98% Accuracy:** Scale-invariant heuristics based on palm-to-finger ratios.
